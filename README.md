## About LaraGram

> **Note:** This repository contains the core code of the LaraGram framework. If you want to build an application using LaraGram, visit the main [LaraGram repository](https://github.com/laraxgram/laragram).

LaraGram is a Telegram Application framework with expressive, elegant syntax. We believe development must be an enjoyable, creative experience to be truly fulfilling. LaraGram takes the pain out of building bots by easing common tasks used in the majority of bot projects, such as:

- [Simple, fast update listening engine](https://laraxgram.github.io/v4/listening).
- [Powerful dependency injection container](https://laraxgram.github.io/v4/container).
- Multiple back-ends for [session](https://laraxgram.github.io/v4/session) and [cache](https://laraxgram.github.io/v4/cache) storage.
- Expressive [conversations](https://laraxgram.github.io/v4/conversations) for multi-step dialogs.
- [Robust background job processing](https://laraxgram.github.io/v4/queues).
- Full [Telegram Bot API](https://laraxgram.github.io/v4/requests) coverage with a fluent request layer.

LaraGram is accessible, yet powerful, providing the tools needed for large, robust bots, TMAs or MTProto Clients. A superb combination of simplicity, elegance, and innovation gives you a complete toolset required to build any bot with which you are tasked.

## Learning LaraGram
